\# Lab3 – Prelucrarea tablourilor bidimensionale în C



Scopul laboratorului este aprofundarea lucrului cu tablouri bidimensionale și dezvoltarea algoritmilor de prelucrare a matricelor: citire, afișare, numărare, sortare și manipulare a datelor.



Sarcina laboratorului (varianta 17):

\- Se dă o matrice cu n linii și m coloane.

\- Coloanele matricei trebuie ordonate în ordine ascendentă după numărul de elemente negative din fiecare coloană.



Laboratorul conține două implementări:

1\. \*\*Rezolvarea proprie\*\*, folosind indexare 1-based și sortare prin interschimbare succesivă.

2\. \*\*Rezolvarea generată de inteligența artificială\*\*, cu indexare standard 0-based, cod compact și comentarii explicative.



Un obiectiv suplimentar al lucrării este analiza comparativă între cele două abordări, evidențiind diferențele de structură, claritate, stil și eficiență.
